# 自己紹介プロダクト LINKARD

<img src="./public/favicon.png" alt="プロジェクトのロゴ" width="100" height="100">

## 概要

このプロジェクトは、ReactとViteを使用して作成されたモバイル対応の自己紹介ページです。バックエンドにはLaravel（API）を使用しています。

## 特徴

- 新規登録、ログイン、ログアウト
- プロフィールの作成、表示、編集、削除
- いいね機能
- QRコードによるフォロー
- フォロー時のグルーピング機能

## 使用技術

- React: ^18.2.0
- Vite: ^4.4.5
- Laravel (backend)
- antd: ^5.9.2
- react-webcam: ^7.1.1
- その他多数
